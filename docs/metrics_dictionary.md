# Словарь метрик

## Ticket count

Количество обращений в выбранном разрезе. используется для оценки нагрузки и сезонности.

## Sla target hours

Целевое время решения обращения. зависит от приоритета и клиентского сегмента.

## Sla breached

Флаг нарушения целевого времени решения. для открытых обращений рассчитывается по текущему возрасту обращения.

## Sla breach rate

Доля обращений с `sla_breached = 1` среди всех обращений в группе.

Формула:

```text
sla breach rate = sla breached tickets / tickets created
```

## First response hours

Количество часов от создания обращения до первого ответа поддержки.

## Resolution hours

Количество часов от создания до закрытия. для открытых обращений показывает текущий возраст на конец периода генерации.

## Backlog open tickets

Количество обращений, которые были созданы, но не закрыты к концу выбранного дня.

## Avg satisfaction score

Средняя оценка клиента по закрытым обращениям. в синтетических данных оценка снижается при длинном решении и нарушении sla.
